# Review And Conversion

## Review workflow
1. Identify the claimed family.
2. Check hierarchy first, not decoration first.
3. Check whether typography, spacing, surfaces, and component treatment tell the same story.
4. Name mismatches as concrete failures.
5. End with the minimum set of changes that would restore fidelity.

## Review questions
- Does the hero express the family in the first screen?
- Does the section rhythm match the family?
- Is the typography doing the right amount of work?
- Are surfaces helping or diluting the style?
- Is motion aligned with the intended intensity?
- Which 2-3 generic patterns are killing fidelity?

## Conversion workflow
1. Preserve content hierarchy and product goals.
2. Remove visual habits from the source family.
3. Rebuild hierarchy, layout rhythm, and component treatment in the target family.
4. Change motion/material language last.
5. Explain trade-offs if conversion will reduce clarity or increase spectacle.

## Conversion prompts should include
- source family
- target family
- elements to preserve
- acceptable motion level
- acceptable experimentation level

## Fast conversion map
- grid-product -> editorial-typography: reduce card count, increase type authority, create breathing room
- grid-product -> experimental-loud: keep priority, break uniformity, harden tension points
- editorial-typography -> high-agency: preserve restraint, add asymmetry and stronger CTA energy
- immersive-premium -> grid-product: reduce atmosphere, keep one focal accent, restore scan clarity
- tactile-organic -> grid-product: keep warmth, remove excess softness, sharpen structure
- clean SaaS -> neo-brutalism: preserve information order, switch to hard edges, louder contrast, stronger headlines

## Reporting format
- Current fidelity
- What is working
- What breaks the family
- Conversion moves
- Risks after conversion
